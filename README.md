## 🇬🇧 English

### What is NotepadRewind?

NotepadRewind is a Windows tool that replaces the new AI-powered Windows 11 Notepad with the classic Windows XP one. No bloat, no AI features, no unnecessary RAM usage — just a simple, fast text editor that gets out of your way.

The new Windows 11 Notepad includes AI features that consume extra RAM and introduce potential security risks derived from AI integration. The classic Notepad does none of that.

### Why use the classic Notepad?

- **Less RAM usage** — the classic Notepad has no AI engine running in the background
- **No AI-related security vulnerabilities** — the new Notepad's AI features increase the attack surface; the classic one doesn't have any
- **Simplicity** — it opens instantly, edits text, and nothing else

### How does it work?

1. Run **NotepadRewind**
2. Select your Windows XP `notepad.exe` file
3. NotepadRewind stores it safely in AppData and sets it as the default app for `.txt` files
4. A shortcut is created in the Start Menu for easy access

### Installation

1. Download `NotepadRewind.exe` from the [Releases](../../releases) section
2. Run `NotepadRewind.exe` as Administrator
3. Select your Windows XP `notepad.exe` when prompted
4. Done!

> ⚠️ **Notice:** NotepadRewind requires a copy of the Windows XP Notepad (`notepad.exe`) that you own legally. NotepadRewind does not distribute any Microsoft software.

## 🇪🇸 Español

### ¿Qué es NotepadRewind?

NotepadRewind es una herramienta para Windows que reemplaza el nuevo Bloc de notas de Windows 11 con IA por el clásico de Windows XP. Sin bloatware, sin funciones de IA, sin consumo innecesario de RAM — simplemente un editor de texto rápido y ligero.

El nuevo Bloc de notas de Windows 11 incluye funciones de IA que consumen RAM adicional e introducen posibles vulnerabilidades de seguridad derivadas de la integración con IA. El clásico no hace nada de eso.

### ¿Por qué usar el Bloc de notas clásico?

- **Menos uso de RAM** — el Bloc de notas clásico no tiene ningún motor de IA ejecutándose en segundo plano
- **Sin vulnerabilidades relacionadas con la IA** — las funciones de IA del nuevo Bloc de notas aumentan la superficie de ataque; el clásico no tiene ninguna
- **Simplicidad** — se abre al instante, edita texto, y nada más

### ¿Cómo funciona?

1. Ejecuta **NotepadRewind**
2. Selecciona tu archivo `notepad.exe` de Windows XP
3. NotepadRewind lo guarda de forma segura en AppData y lo establece como aplicación predeterminada para los archivos `.txt`
4. Se crea un acceso directo en el Menú de inicio para un acceso rápido

### Instalación

1. Descarga `NotepadRewind.exe` desde la sección [Releases](../../releases)
2. Ejecuta `NotepadRewind.exe` como Administrador
3. Selecciona tu `notepad.exe` de Windows XP cuando se te pida
4. ¡Listo!

> ⚠️ **Aviso:** NotepadRewind requiere una copia del Bloc de notas de Windows XP (`notepad.exe`) que tengas legalmente. NotepadRewind no distribuye ningún software de Microsoft.
